# 📈 XAUUSD AI Swing Trading Dashboard

Selamat datang di repository proyek Dashboard Swing Trading untuk XAUUSD (Gold) berbasis Artificial Intelligence (AI).

Aplikasi ini dibangun menggunakan **Python** dan **Streamlit** untuk memvisualisasikan data pasar, menganalisis tren, dan memberikan sinyal trading secara real-time yang terintegrasi dengan MetaTrader 5 (MT5).

## ✨ Fitur Utama
- 📊 **Visualisasi Chart:** Menampilkan pergerakan harga XAUUSD secara interaktif.
- 🤖 **Sinyal AI:** Memberikan rekomendasi Entry, Take Profit, dan Stop Loss berdasarkan model Machine Learning.
- 📉 **Indikator Teknikal:** Menampilkan perhitungan otomatis untuk RSI, MACD, Moving Average, dll.
- 📈 **Tracking Performa:** Memantau Win/Loss ratio dan riwayat trading.

## 🛠️ Tech Stack
- **Bahasa Pemrograman:** Python 3.9+
- **Frontend / UI:** Streamlit
- **Data & AI:** Pandas, NumPy, Scikit-Learn (atau TensorFlow/PyTorch)
- **Broker Integration:** MetaTrader 5 (MT5) Python API
